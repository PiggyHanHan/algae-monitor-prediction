### 无人机藻类识别项目 - AI视觉（Tianyu）专属 README

#### 一、核心定位
作为项目AI视觉算法负责人，聚焦藻类视觉识别全流程。核心职责包括：
- **提供统一预处理脚本**：集成画质修复 + 几何校正（正射校正、统一地面分辨率），供贺一冉处理训练数据，也用于用户上传图片的推理前处理；
- 基于贺一冉交付的标准化预处理图像，完成藻类像素级定位、分类、量化；
- 输出标准化结果（含原始元数据 + 已计算藻类绝对面积）至数模组。

不参与数据采集、初始标注、数模预测等非视觉算法环节。

#### 二、绝对清晰的工作边界

✅ **核心负责（AI视觉全流程）**
1. 编写并提供统一预处理脚本（画质修复+几何校正，统一地面分辨率和尺寸）
2. 基于贺一冉输出的预处理干净图像，开展数据集标注复核
3. 搭建、训练、调优U-Net藻类语义分割模型（像素级定位+多类别区分）
4. 执行模型推理：识别藻类有无、位置、门类（蓝藻/绿藻/其他）
5. 量化计算：基于统一地面分辨率，直接计算**藻类绝对面积（平方米）**、各门类面积
   **注意：不计算藻类占整个池塘的百分比，不输出浓度等级，不输出图像覆盖面积。**
6. 输出标准化识别结果（含原始元数据）至数模组

❌ **绝不涉及（归数据/航拍/数模模块）**
1. 不参与任何原始图像采集、预处理（画质修复+几何校正由统一脚本完成，但脚本开发归AI模块）
2. 不使用LabelMe开展初始像素级标注（仅复核）
3. 不进行时序分析、预测建模等数模环节工作
→ 以上全部归：贺一冉 + 航拍组 + 数模组

#### 三、分步执行任务（按顺序推进）

1. **统一预处理脚本开发（✅ 已明确）**
   - 编写预处理脚本，集成：
     - 画质修复：去反光、去雾、色彩校正、畸变矫正
     - 几何校正：根据无人机高度、相机参数（焦距、传感器尺寸）及姿态角（俯仰、滚转、偏航），将图像校正为正射视角，**统一地面分辨率**（例如 0.02 米/像素）
       - **执行顺序**：先几何校正为正射图 → 计算实际GSD → 重采样至目标GSD → 不再额外缩放
     - 若缺少姿态角，默认按垂直拍摄处理并记录警告；若缺少高度或焦距，脚本终止并提示补充元数据。
   - 脚本需支持批量处理（供贺一冉使用）和单张处理（供推理使用）
   - 输出：可独立运行的预处理脚本 + 使用说明文档
   - **维护责任**：AI模块负责脚本的维护、bug修复与版本升级；数据端仅作为使用者。

2. **数据接收与核验（✅ 已明确核验标准）**
   - 接收贺一冉交付的：**经统一脚本预处理后的干净图像集**、LabelMe标注文件（JSON）、掩码图、元数据台账（CSV）
   - 核验内容：
     - 图像完整性：可读、尺寸统一（2048×2048）、格式PNG
     - 标注一致性：随机抽取10%标注文件，目测边界准确；统计各类别像素比例，异常（如阴影类占比>50%）人工复核
     - 元数据完整性：台账字段齐全（见元数据规范），值合理
     - 交付结构：文件夹结构符合约定
   - 核验不通过时，提供错误报告，数据端需在3个工作日内修正
   - 输出：核验通过的标准化训练数据集

3. **模型搭建与训练（✅ 已明确）**
   - 基于U-Net架构搭建语义分割模型，适配标注数据格式
   - 划分训练/验证/测试集，调优超参数
   - 训练目标：精准区分藻类（蓝/绿/其他）、清水、泥沙、阴影
   - 输出：训练完成且验证达标的藻类分割模型

4. **模型推理与量化计算（✅ 已明确）**
   - 输入：**经统一脚本预处理的单张图像**
   - 输出：
     - 藻类像素级定位掩码图（区分门类）
     - 基于统一地面分辨率，直接计算：
       - 各藻类真实面积（平方米）
       - 整体藻类总面积（平方米）
       - 各类型藻占比（面积占比）
     - **不输出浓度等级，不输出图像覆盖面积**
   - 量化结果在AI模块完成，数模模块无需二次计算

5. **结果交付与对接（✅ 已明确输出格式）**
   - 按标准接口整理识别结果，生成 JSON 文件（结构见下方）
   - 交付至数模组，不参与后续分析/预测环节
   - 输出：标准化交付结果包（含分割掩码图PNG、JSON结果）

#### 四、模块分工（一句话界定）
- 航拍组：采集原始无人机图像
- 贺一冉：原始图像→使用统一脚本预处理→LabelMe标注数据集
- 本人：提供统一脚本 + 标注数据集→U-Net模型训练→藻类识别/量化结果（含JSON输出）
- 数模组：识别量化结果→时序分析/预测建模

#### 五、最终交付物
1. 训练调优完成的U-Net藻类语义分割模型文件（含权重）
2. **统一预处理脚本**（含几何校正功能）及使用说明文档
3. 模型推理代码（可独立运行，输出 JSON 格式）
4. 藻类识别可视化结果图（掩码叠加原图）
5. **JSON 输出接口文档**（字段说明、数据格式规范）
6. 数模组对接的标准输出接口文档
