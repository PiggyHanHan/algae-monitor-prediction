### 无人机藻类识别项目 - 数据链路（贺一冉）专属 README

#### 一、核心定位
作为项目全链路数据负责人，全权把控数据从原始采集到标准化标注的全流程。**使用 AI 视觉模块提供的统一预处理脚本**对原始图像进行画质修复 + 几何校正，输出唯一、高质量的标准化藻类数据集，为AI视觉模块提供可直接用于U-Net训练/推理的标准化数据输入，不参与模型相关工作。

#### 二、绝对清晰的工作边界

✅ **核心负责（数据端全流程）**
1. 配合航拍组完成无人机图像采集，严格把控拍摄规范与原始数据归档，并记录无人机元数据（高度、视角、GPS等）
2. 使用 AI 视觉模块提供的统一预处理脚本，对原始图像执行全流程预处理（画质修复+几何校正，统一地面分辨率和尺寸）
3. 使用LabelMe完成像素级精准标注，生成标准化标注文件与掩码图
4. 归档全量数据，交付至AI视觉模块并配合完成数据核验

❌ **绝不涉及（归AI视觉+数模模块）**
1. 不搭建、训练、推理U-Net语义分割模型
2. 不计算藻类面积、覆盖率等量化指标
3. 不参与模型结果对接、时序分析、预测建模
→ 以上全部归：AI视觉模块（Tianyu+刘俊辉）+ 数模组

#### 三、分步执行任务（按顺序推进，不跳步）

1. **原始数据采集与归档（✅ 已明确）**
   - 对接航拍组，明确拍摄规范：垂直90°俯拍、M手动档、RAW格式、10-50m高度、无强光反光
   - 收集所有原始航拍图像，按「拍摄时间+区域」分类归档
   - **记录无人机元数据台账**：CSV格式，包含每张图的`image_id`, `timestamp`, `height_m`, `pitch_deg`, `roll_deg`, `yaw_deg`, `gps_lat`, `gps_lon`, `camera_focal_mm`, `sensor_width_mm`, `sensor_height_mm`
   - 输出：无缺失、无格式错误的原始无人机航拍图像集 + 元数据台账CSV

2. **图像全流程预处理（✅ 已明确）**
   - **使用 AI 视觉模块提供的统一预处理脚本**，对原始图像批量处理：
     - 画质修复：去除水面高光反光、雾感/噪点、色彩校正、对比度增强、镜头畸变矫正
     - 几何校正：根据无人机高度、相机参数及姿态角将图像校正为正射视角，**统一地面分辨率**（例如 0.02 米/像素）
       - **执行顺序**：先几何校正为正射图 → 计算实际GSD → 重采样至目标GSD → 不再额外缩放
     - 尺寸统一：缩放到固定尺寸（例如 2048×2048）
   - 输出：无反光、无杂色、藻类边界清晰、空间尺度统一的预处理干净图像集
   - **脚本维护**：AI模块负责脚本维护；若运行遇到问题，及时反馈AI模块修复。

3. **LabelMe像素级标注（✅ 已明确）**
   - 统一标注类别：蓝藻、绿藻、其他藻类、清水、泥沙、阴影（英文标签对应`blue_algae`, `green_algae`, `other_algae`, `clear_water`, `sediment`, `shadow`）
   - 手动精准绘制藻类像素边界，无漏标/错标/重复标注
   - 生成JSON格式标注文件 + 对应像素掩码图（Mask）
   - 输出：标注完成的标准化数据集（干净原图+JSON标注+掩码图）

4. **数据交付与核验配合（✅ 已明确核验标准）**
   - 向AI视觉模块（Tianyu+刘俊辉）交付全量标注数据集，文件夹结构如下：
     ```
     dataset/
       images/          # 预处理干净图像
       annotations/     # LabelMe JSON
       masks/           # 像素掩码图
       metadata.csv     # 元数据台账
     ```
   - 配合AI模块完成数据核验（核验标准见AI模块README）
   - 若核验不通过，3个工作日内修正并重新交付
   - 输出：验收通过的最终交付数据集

#### 四、模块对接（唯一交付对象）
- 对接对象：AI视觉模块（Tianyu+刘俊辉）
- 交付内容：
  1. 预处理干净图像集（统一格式、尺寸、地面分辨率）
  2. LabelMe标注文件（JSON）+ 像素掩码图（Mask）
  3. 数据归档文档（采集规范/预处理参数/标注规则）
  4. 无人机元数据台账CSV
- 对接要求：交付前完成全量数据自检，确保无损坏、无标注错误，AI视觉端可直接读取使用

#### 五、最终交付物
1. 分类归档的完整原始航拍图像集（含采集信息台账）
2. **无人机元数据台账CSV文件**
3. **使用统一脚本处理后的干净图像集**（标注/模型可用，附预处理参数记录）
4. LabelMe标注完成的全量数据集（干净原图+JSON标注+掩码图）
5. 数据链路全流程文档（采集规范、预处理手册、标注规则）
6. 数据交付核验单（AI视觉模块确认签字版）
